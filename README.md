# Habit Tracker 

Веб-приложение для отслеживания и формирования полезных привычек. Помогает пользователям ставить цели, регулярно отмечать прогресс и анализировать свою продуктивность с помощью наглядной статистики.

##  Ключевые функции

-  **Регистрация и авторизация** — безопасный вход в систему
-  **Создание привычек** — настройка индивидуальных целей и расписания
-  **Отметка выполнения** — быстрая фиксация прогресса каждый день
-  **Календарь** — визуальное отображение дней выполнения
-  **Статистика** — аналитика прогресса и успешности привычек

## Стек технологий

- **Backend:** Python
- **Frontend:** React + TypeScript
- **Database:** PostgreSQL
