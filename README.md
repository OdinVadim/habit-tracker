# Habit Tracker 

Веб-приложение для отслеживания и формирования полезных привычек.

## Стек технологий

- **Backend:** Python
- **Frontend:** React + TypeScript
- **Database:** PostgreSQL

##  Структура проекта
├── backend/ # Серверная часть на Python
├── frontend/ # Клиентская часть на React
├── .gitignore # Правила для Git
├── README.md # Описание проекта
└── plan.txt # План разработки
