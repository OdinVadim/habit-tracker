План разработки Habit Tracker
==============================

Этап 1: Настройка окружения
----------------------------------------------
1.1. Настройка репозитория (.gitignore, README, plan.txt)
1.2. Проектирование схемы БД (PostgreSQL)
     - Таблицы: users, habits, habit_logs
1.3. Инициализация Backend (FastAPI проект)
1.4. Инициализация Frontend (React + TypeScript + Vite)

Этап 2: Backend - Базовая функциональность
----------------------------------------------
2.1. Подключение к БД (SQLAlchemy + asyncpg)
2.2. Модели данных (User, Habit, HabitLog)
2.3. Регистрация и авторизация (JWT токены)
2.4. CRUD для привычек (создание, чтение, обновление, удаление)
2.5. Написание тестов для API (Pytest)

Этап 3: Frontend - Базовая функциональность
----------------------------------------------
3.1. Настройка роутинга (React Router)
3.2. Страницы: Login, Register, Dashboard
3.3. Сетевой модуль (Axios) для связи с API
3.4. Управление состоянием (Zustand / Redux Toolkit)
3.5. Формы создания и редактирования привычек

Этап 4: Основной функционал
----------------------------------------------
4.1. Отметка выполнения привычки (чекбокс/кнопка)
4.2. Календарь прогресса (отображение дней выполнения)
4.3. Статистика (Recharts / Chart.js)
4.4. Фильтрация и сортировка привычек
4.5. Адаптивный дизайн (Mobile friendly)

Этап 5: Деплой и Финализация
----------------------------------------------
5.1. Настройка CI/CD (GitHub Actions)
5.2. Деплой Backend (Render / Railway / VPS)
5.3. Деплой Frontend (Vercel / Netlify)
5.4. Финальное тестирование и багфикс
5.5. Написание документации для пользователя
